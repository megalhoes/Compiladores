# NovaLang — Compilador Basico

Compilador educacional em Python — Disciplina de Compiladores, UNINASSAU.

## Como Executar

```bash
python main.py exemplos/exemplo1.nova
python main.py exemplos/exemplo2.nova
python main.py exemplos/exemplo3.nova   # deve imprimir 17
python main.py                          # REPL interativo
```

- Funcoes com parametros e retorno
- print() e input()
