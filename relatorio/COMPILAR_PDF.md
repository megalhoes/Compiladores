# Como Gerar o Relatorio PDF

## Requisitos
- MikTeX: miktex.org
- TeXmaker: xm1math.net/texmaker

## Passos
1. Abrir TeXmaker > File > Open > `relatorio.tex`
2. Pressionar **F1** (PDFLaTeX)
3. Pressionar **F1** novamente (resolve sumario e referencias)
4. O `relatorio.pdf` é gerado na mesma pasta

*.aux  *.log  *.toc  *.out
